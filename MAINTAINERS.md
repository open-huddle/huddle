# Maintainers

This file lists the current maintainers of Open Huddle. Maintainers have merge rights on this repository and are responsible for upholding its technical, security, and compliance standards.

See [GOVERNANCE.md](GOVERNANCE.md) for what maintainers do, how decisions are made, and how new maintainers are added.

## Becoming a maintainer

See the **Promotion criteria** for Maintainers in [GOVERNANCE.md](GOVERNANCE.md#maintainers). In short: sustained, high-quality contribution over at least six months, nomination by an existing maintainer, and approval by two-thirds of the current maintainers.

## Contact

For general project questions, use [GitHub Discussions](https://github.com/open-huddle/huddle/discussions). For security issues, see [SECURITY.md](SECURITY.md). For conduct issues, email conduct@open-huddle.org.
